# 🎬 IMDb Movie Rating Predictor

A production-ready machine learning web application that predicts IMDb ratings for Indian movies based on genre, cast, director, release year, duration, and vote count.

---

## 🚀 Quick Start

### 1. Clone & Install

```bash
git clone https://github.com/your-username/movie-rating-prediction.git
cd movie-rating-prediction
pip install -r requirements.txt
```

### 2. Place Dataset

Copy `IMDb Movies India.csv` into the `data/` folder:

```
data/
└── IMDb Movies India.csv
```

### 3. Train the Models

```bash
python train.py
```

This will:
- Clean and preprocess the data
- Engineer features (frequency encoding, log-transform, genre OHE)
- Train 4 regression models
- Save the best model + artefacts to `models/`
- Generate EDA plots in `assets/`

### 4. Launch the Web App

```bash
streamlit run app.py
```

Open http://localhost:8501 in your browser.

## 🧠 Models Trained

| Model | Notes |
|---|---|
| Linear Regression | Baseline |
| Decision Tree | Depth-limited to prevent overfit |
| Random Forest | 200 estimators, parallel training |
| Gradient Boosting | Slow but powerful |

The best-performing model (by R²) is automatically selected and saved.

---

## 📊 Features Used

| Feature | Engineering |
|---|---|
| Genre | Frequency encoding + Top-15 one-hot |
| Director | Frequency encoding |
| Actor 1/2/3 | Frequency encoding |
| Votes | Log1p transform |
| Year | Raw + Movie_Age derived |
| Duration | Raw numeric (minutes) |
| Genre_Count | Count of genres listed |

## 🌐 Deployment

### Streamlit Community Cloud (Free)
1. Push repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your repo → set `app.py` as entry point
4. Deploy — Streamlit handles the rest

### Render
```yaml
# render.yaml
services:
  - type: web
    name: movie-rating-predictor
    env: python
    buildCommand: pip install -r requirements.txt && python train.py
    startCommand: streamlit run app.py --server.port $PORT --server.address 0.0.0.0
```

### Railway
Create a `Procfile`:
```
web: streamlit run app.py --server.port $PORT --server.address 0.0.0.0
```
